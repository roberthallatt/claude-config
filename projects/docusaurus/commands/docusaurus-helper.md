# Docusaurus Helper Command

Provides assistance with Docusaurus documentation site development.

## Development Commands

```bash
# Start development server
npm run start

# Start with specific port
npm run start -- --port 3001

# Build for production
npm run build

# Serve production build
npm run serve

# Clear cache
npm run clear

# Type check
npm run typecheck
```

## Content Structure

### Documentation (`docs/`)

```markdown
---
id: page-id
title: Page Title
sidebar_label: Sidebar Label
sidebar_position: 1
description: SEO description
tags: [tag1, tag2]
---

# Page Content

Your documentation here...
```

### Blog Posts (`blog/`)

```markdown
---
title: Blog Post Title
authors: [author-key]
tags: [tag1, tag2]
date: 2024-01-15
---

Blog post content...

<!--truncate-->

More content after the fold...
```

## Sidebar Configuration

### Auto-generated
```javascript
// sidebars.js
module.exports = {
  docs: [
    {
      type: 'autogenerated',
      dirName: '.',
    },
  ],
};
```

### Manual
```javascript
module.exports = {
  docs: [
    'intro',
    {
      type: 'category',
      label: 'Getting Started',
      items: ['installation', 'configuration'],
      collapsed: false,
    },
  ],
};
```

### Category Metadata
```json
// docs/category/_category_.json
{
  "label": "Category Name",
  "position": 2,
  "collapsible": true,
  "collapsed": true
}
```

## Versioning

```bash
# Create new version
npm run docusaurus docs:version 1.0.0

# Structure after versioning
docs/           # Next/unreleased version
versioned_docs/
  version-1.0.0/
versioned_sidebars/
  version-1.0.0-sidebars.json
versions.json
```
