# Brand Color System

## Primary Colors

### Brand Green (#238937)
- Primary action color
- Navigation backgrounds
- Hero sections
- CTA buttons
- Use: `bg-brand-green`, `text-brand-green`, `border-brand-green`

### Brand Blue (#00639A)
- Secondary actions
- Links
- Accents
- Use: `bg-brand-blue`, `text-brand-blue`

## Accent Colors

### Brand Orange (#F15922)
- Highlights
- Important notices
- Attention-grabbing elements
- Use: `bg-brand-orange`, `text-brand-orange`

### Brand Light Green (#D7DF21)
- Light accents
- Hover states
- Background highlights
- Use: `bg-brand-light-green`, `text-brand-light-green`

## Usage Guidelines

### Buttons
- Primary CTA: `bg-brand-green text-white hover:bg-opacity-90`
- Secondary: `border-2 border-brand-blue text-brand-blue`
- Alert: `bg-brand-orange text-white`

### Links
- Navigation: `text-brand-blue hover:text-brand-green`
- Hover effect: `text-brand-blue underline hover:text-brand-green`

### Backgrounds
- Sections: `bg-brand-green text-white` or `bg-gray-100`
- Cards: `bg-white border-l-4 border-brand-green`
- Overlays: `bg-black/50` or `bg-brand-green/10`

### Text Hierarchy
- Headings: Use `text-brand-green` for emphasis
- Body: Use `text-gray-800`
- Muted: Use `text-gray-600`

## Color Combinations

### Dark Background + Light Text
```
bg-brand-green text-white
bg-brand-blue text-white
```

### Light Background + Dark Text
```
bg-brand-light-green text-gray-900
bg-gray-100 text-brand-green
```

### Accent Pairs
```
bg-white border-l-4 border-brand-orange
bg-brand-green text-white hover:bg-brand-orange
```
